# Bob_Lex
A cute selfmade Redo of the Bopcat steam desktop companion. because i wanted to make it with my own character. for now verry basic might be updated in the future.


bob_lex.exe V 1.1 READ ME

To change Size hover over window and scroll with Mouse.

To change Texture: go into Texture Folder and replace current Textures with your own. 
Mouth is not currently used. 
Make sure to give youre new textures the name of the old ones and that the old ones are named something else or deleted. 
Make sure new texture is PNG. and has PNG Label. 
have fun. <3
